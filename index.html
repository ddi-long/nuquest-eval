키 발급 방법 (5분, 무료, 신용카드 불필요)
1. aistudio.google.com/app/apikey 접속
2. 구글 계정 로그인 → 우측 상단 "Create API key" 클릭
3. AIzaSy로 시작하는 문자열 복사 → 위 입력란에 붙여넣기 → "저장"
한도: 일 약 1,500회 무료 (Gemini 2.5 Flash)
