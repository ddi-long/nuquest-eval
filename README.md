# NUQUEST 자동 평가 도구

RCT 논문(PDF)을 업로드하면 NUQUEST 14개 항목에 대한 평가 결과를 자동으로 생성하는 웹 도구입니다.

## 사용 방법

1. https://ddi-long.github.io/nuquest-eval/ 접속
2. Gemini API 키 입력 (최초 1회, 본인 브라우저에만 저장됨)
3. RCT 논문 PDF 업로드 (최대 5개)
4. 결과 확인 및 엑셀 다운로드

## Gemini API 키 발급 (무료, 5분)

1. https://aistudio.google.com/app/apikey 접속
2. 구글 계정 로그인 → "Create API key" 클릭
3. `AIzaSy`로 시작하는 키 복사 → 본 도구의 API 키 입력란에 붙여넣기

신용카드 불필요, Gemini 2.5 Flash 무료 티어 사용 (분당 약 15회, 일 약 1,500회).

## 주의사항

⚠️ **무료 티어는 Google이 입력 내용을 모델 개선에 활용할 수 있습니다.**
미발표 논문, 동료심사 중 원고, 기밀 자료의 평가에는 사용을 권장하지 않습니다.
이미 출판된 논문 평가에만 사용하세요.

## 적용 대상

- 적용 가능: RCT (parallel-arm, crossover, acute randomized feeding trial)
- 적용 불가: 비무작위 중재연구, single-arm, observational study

## 스캔본(이미지) PDF

- 텍스트 레이어가 없는 스캔본은 **자동 감지**되어 Gemini가 PDF 원본을 직접 **OCR**로 읽어 평가합니다.
- 디지털 원문 PDF라도 **표·그림까지 읽히고 싶으면** 파일 목록의 `원본(표·그림) 판독`을 체크하세요.
- 스캔본은 대조할 텍스트 레이어가 없어 **근거 검증(신호등)은 생략**됩니다 — 발췌 근거를 직접 확인하세요.

## 항목별 판정 (5단계 표시 / 3단계 집계)

NUQUEST 공식 척도(YES / PROBABLY YES / PROBABLY NO / NO / NOT APPLICABLE)에 따라 5개 라벨로 표시합니다.
점수 집계 시에는 **아마도 예 → 예**, **아마도 아니오 → 아니오**로 환산합니다.

| 판정 | 의미 |
|---|---|
| 예 / 아마도 예 | 기준 충족(확실 / 부분·검토 권장) |
| 아니오 / 아마도 아니오 | 기준 미충족(확실 / 부분·검토 권장) |
| 적용되지 않음 | 해당 항목이 연구에 해당하지 않음 |

## 검토자(사용자) 평가

AI 판정 옆/아래에 검토자가 **자신의 판정**을 선택하고 **메모**를 남길 수 있습니다.
AI 판정과의 일치 여부(✓ 일치 / ≈ 경미한 차이 / ✗ 불일치)가 표시되며, 입력 내용은 엑셀로 함께 내려받을 수 있습니다.
입력은 **브라우저에 자동 저장**되어 새로고침/재평가해도 유지됩니다(같은 파일명 기준).
(독립 평가자 간 결과 정합성 검토·논의를 지원 — AI 판정은 그대로 보존)

## 엑셀 내려받기

- **논문당 1시트** + 전체 요약 `results` 시트로 구성됩니다.
- 각 논문 시트: 14개 항목(AI 판정·근거·검토자 판정·메모), 영역별 종합, 총점 A/B.
- 병합셀 없이 표 형태로 정리되며, 긴 텍스트는 자동 줄바꿈됩니다.

## 근거 검증 신호등

각 판정의 발췌 문장이 PDF 원문에 실제로 존재하는지(hallucination 점검) 자동 대조합니다.

| 표시 | 의미 |
|---|---|
| 🟢 | 발췌 문장이 PDF 원문에 일치 — 신뢰 가능 |
| 🟡 | 유사 매칭이거나 검토 필요 — 사용자 확인 권장 |
| 🔴 | 발췌 문장 PDF 미발견 — hallucination 의심 |
| ⚪ | 적용되지 않음 |

## 최종 총점 (2종 동시 산출)

- **총점 A — 농산물 특성 반영**: 영양특화(영역4) 항목의 가중치를 낮춘 결과.
- **총점 B — NUQUEST 원 기준**: 4개 영역을 동일 가중하여 산출한 결과.

각 총점은 우수(바이어스 위험 낮음) / 보통(중간) / 부족(높음)으로 표시됩니다.

## 라이선스 / 데이터

- API 키와 PDF는 사용자 브라우저에만 머무르며, GitHub 또는 본 도구 운영자에게 전송되지 않습니다.
- Gemini API에는 PDF 텍스트가 전송됩니다 (Google 무료 티어 약관 적용).
